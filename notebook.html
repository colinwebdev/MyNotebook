My Notebook

Instructions

This is a simple little page that uses your browser's cache to store information. Nothing is sent the server, and only you can access what's stored here.

The form accepts 3 different types of input: images via URL, links, and text input.

To include an image or link, just paste the URL into the input box and the page takes care of the rest. You may include additional text with your link or image, but all images will be stored together regardless of where they are added in the text.

For text, you can type in as much or as little as you want. But only plain text can be used, no HTML is allowed.

Use the delete button on the item's card to remove the item, or use the edit button to make changes.

If you want to wipe all information, use the "Clear All" button at the bottom of the page. This will remove all cards.
